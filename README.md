# Hell’s Kitchen Restaurant Website
A responsive restaurant website built using HTML, CSS, JavaScript, and Bootstrap.

## Project Overview
Hell’s Kitchen is a modern and interactive restaurant website designed to provide users with information about the menu, facilitate table reservations, and allow users to contact the restaurant easily. The website is built using **HTML, CSS, JavaScript**, and **Bootstrap** for responsive layout and interactive components.

## Project Objectives
- Create a visually appealing and professional restaurant website.
- Enable users to browse the menu with categorized interactive item cards.
- Allow table reservations through a user-friendly form.
- Provide an easy way for customers to contact the restaurant.
- Enhance user experience with animations and interactive UI features.

## Pages Description

### Home Page (`index.html`)
- Hero section with logo and tagline.
- Counter section with animated statistics.
- Footer with copyright.

### Menu Page (`menu.html`)
- Categories: Steaks, Pasta, Desserts.
- Interactive menu cards with:
  - Image, description, price, rating.
- Cinematic background images with overlay effect.

### Reservation Page (`reservation.html`)
- Reservation form with fields:
  - Full Name, Email, Phone, Date, Time, Number of Guests, Special Requests.
- Confirmation alert on submission.

### About Page (`about.html`)
- Hero section with gradient overlay.
- Mission & Vision sections.
- Chef section with highlights.
- Features/info cards for restaurant services.

### Contact Page (`contact.html`)
- Hero section with restaurant background.
- Contact info cards (phone, email, address).
- Contact form with success alert on submission.

## CSS Overview
- `style.css` for custom styles:
  - Navbar, hero sections, menu cards, reservation and contact forms.
- `bootstrap.min.css` for responsive layout, grid system, and utilities.

## JavaScript Overview
- `script.js`:
  - Navbar scroll effect.
  - Reservation and contact form alerts and reset.
  - Counter animations on the home page.
- `bootstrap.bundle.min.js`:
  - Bootstrap interactive components (modals, dropdowns, responsive navbar).

## How to Run
1. Open the project folder.
2. Open `index.html` in a modern browser.
3. Navigate pages using the top navigation bar.
